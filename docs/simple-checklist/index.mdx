# Simple Checklist For Jira
**Simple Checklist for Jira** is an application that revolutionizes your task management experience by providing an effortless way to create and manage checklists within Jira.

## Installation

You can install **Simple Checklist for Jira** from the [Atlassian Marketplace](https://marketplace.atlassian.com/apps/1231675/simple-checklist-for-jira?hosting=cloud&tab=overview).

## Easy Access
To access your checklists, follow these simple steps:
1. Navigate to the Jira issue you're interested in managing.
2. Look for the checklist icon in the menu.
3. Click the checklist icon to open and access your checklists.

## Features

- [X] Checklist items within Jira issues.
- [X] Easy to create and manage checklist items.
- [X] No complicated forms to fill out.
- [X] Track progress and completion of checklist items.
